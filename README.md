# Front End Setup #

This is a ready setup to build front end application only. No plugin included

### What is this repository for? ###

* Offical repository for front end works
* Version 1.0

### How do I get set up? ###

* Run npm install in cli
* Run npm start

### What is in here? ###
* SCSS / SMACSS
* NPM Gulp
* NPM Gulp Sass
* NPM Clean CSS
* NPM Beautify CSS
* NPM Lite Server (BrowserSync)
* NPM Concurrently
